Typographie

En utilisant l'unité rem dans le framework, nous avons créé une échelle typographique qui peut être modifié en changeant simplement la taille de l'unité de base dans l'élément root. La typographie actuelle a une base de 1.6rem soit 16px. Chaque lineheight est relative et va de 125% à 150% en prenant la font-size du texte comme base. Le Turtle System utilise la font-family Poppins, créé par Jonny Pinhorn et fourni à travers le service Google Fonts.

Grid

Grid est un outil puissant, permettant de gérer des affichages en colonnage simplement. Pour le framework Turtle System, nous avons opté pour un système de grille flexible mettant à profit les unités fr, la fonction repeat ainsi les attributs minmax() et auto-fit . Cela permet de gérer l'adaptation aux différents supports sans media queries dans la grande majorité des cas. Ainsi, pour mettre un container en display : grid;, il suffit de lui attriber la class .grid et il hérite de la stylisation suivante : 

.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  grid-gap: var(--m);
}
